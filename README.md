# 🚀 FounderOS — AI Chief of Staff

FounderOS is an AI-powered operational system for startup founders. It bridges the gap between fragmented workspace data (Gmail, Slack, Docs) and high-level strategic execution.

## 🌟 Key Highlights
- **AI Chief of Staff**: Real-time page analysis and summary.
- **Magic Sync**: Zero-friction background data synchronization from the browser.
- **Deadline Radar**: Automated task extraction from your emails.
- **Investor Pipeline**: AI-detected leads and follow-up tracking.

## 🚀 Quick Start

### 1. Dashboard
```bash
npm install
npm run dev
```
Visit `http://localhost:3000`

### 2. Chrome Extension
1. Go to `chrome://extensions/`
2. Enable **Developer Mode**
3. **Load unpacked** -> select `chrome-extension` folder
